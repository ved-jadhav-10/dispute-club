# Character Image Assets

Place 1000x1000 PNG character portraits in this folder.

Filename format (required):
- socrates.png
- napoleon.png
- newton.png
- gandhi.png
- shakespeare.png
- confucius.png

The debate UI loads images from:
`/assets/characters/<figure-id>.png`

If an image is missing, the panel still renders with the placeholder background.
